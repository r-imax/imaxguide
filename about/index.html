About This Project

Open-source IMAX theatre database

Project Mission

The IMAX Theatre Database is a community-driven project providing accurate, searchable information about IMAX theatres worldwide. We maintain technical specifications for screen sizes, projector types, and theatre capabilities.

History: A comprehensive IMAX theatre list was previously available on Wikipedia but was removed. This site exists to provide a freely accessible database with enhanced search and filtering capabilities.

How to Contribute

This project relies on community contributions. You can help in several ways:

Submit Theatre Data

Know of a missing IMAX theatre? Submit complete information including location, screen size, and projector details.

Report Corrections

Found incorrect information? Help us maintain accuracy by reporting errors or outdated details.

Technical Contributions

Developers can contribute code improvements, new features, or help with data processing on GitHub.

Data Submissions

Send theatre data or corrections via email to updates[AT]143190.xyz with detailed information and sources.
